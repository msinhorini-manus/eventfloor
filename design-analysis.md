# Análise de Identidade Visual - ERP Summit

## Paleta de Cores

### Cores Principais
- **Azul Escuro/Navy**: `#0a1628` (fundo principal, header)
- **Laranja/Dourado**: `#f59e0b` / `#ff8c00` (botões primários, CTAs, destaques)
- **Branco**: `#ffffff` (texto principal em fundos escuros)

### Cores Secundárias (bordas dos menus)
- **Ciano**: `#00bcd4` / `#06b6d4` (Home)
- **Roxo/Magenta**: `#9333ea` / `#a855f7` (Exhibitors)
- **Roxo Escuro**: `#7c3aed` (Blog)
- **Laranja**: `#f97316` (Event Dates)
- **Verde**: `#10b981` / `#059669` (Online)

### Gradientes
- Fundo hero: Gradiente diagonal azul escuro com linhas geométricas
- Efeito de profundidade com sobreposições azuis

## Tipografia

- **Fonte Principal**: Sans-serif moderna (similar a Inter, Poppins ou Montserrat)
- **Títulos**: Bold, grandes, brancos em fundos escuros
- **Corpo**: Regular, legível, alto contraste

## Elementos de Design

### Navegação
- Header fixo com fundo azul escuro (`#0a1628`)
- Menu items com bordas coloridas (2-3px) em cores vibrantes
- Logo ERP Summit com moldura dourada

### Botões
- **Primário**: Laranja/dourado (`#f59e0b`) com texto branco
- **Hover**: Efeito de brilho/saturação
- Bordas arredondadas (rounded-lg)

### Cards e Containers
- Bordas brancas arredondadas em fundos escuros
- Sombras suaves para profundidade
- Espaçamento generoso

### Formulários
- Inputs com bordas coloridas (rosa/vermelho para erro, verde para sucesso)
- Labels com cores vibrantes
- Campos com fundo claro em containers escuros

## Estilo Geral

- **Tema**: Dark mode predominante
- **Contraste**: Alto contraste (texto branco em fundo azul escuro)
- **Modernidade**: Design moderno com elementos geométricos
- **Energia**: Cores vibrantes e saturadas para CTAs e destaques
- **Profissionalismo**: Layout limpo e organizado

## Aplicação na Plataforma

1. Mudar tema de light para dark
2. Aplicar azul escuro como cor de fundo principal
3. Usar laranja/dourado para botões primários e CTAs
4. Adicionar bordas coloridas nos elementos de navegação
5. Implementar gradientes azuis no hero
6. Aumentar contraste geral
